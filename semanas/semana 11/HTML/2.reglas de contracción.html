Reglas de Contracción

Verbos Contractos en Griego

Reglas de Contracción

1. La contracción ου

Se forma a partir de:

  • εο → ου
    Ejemplo: ποιοῦμεν ← ποιε-ομεν
  • οε → ου
    Ejemplo: πληροῦτε ← πληρο-ετε
  • οο → ου
    Ejemplo: πληροῦμεν ← πληρο-ομεν
2. La contracción ει

Se forma a partir de:

  • εε → ει
    Ejemplo: ποιεῖτε ← ποιε-ετε
3. La contracción ω

Se forma casi con cualquier combinación de omicrón (ο) u omega (ω) con otra vocal (excepto las de la regla 1):

  • αο → ω
    Ejemplo: ἀγαπῶμεν ← ἀγαπα-ομεν

Nota especial: En la primera persona singular del presente, el verbo contracto aparece en su forma léxica con la vocal contracta (ej. ἀγαπάω), pero en el texto bíblico se contrae (ej. ἀγαπῶ).

4. La contracción α (ᾳ)

Se forma a partir de:

  • αε → α
    Ejemplo: ἀγαπᾶτε ← ἀγαπα-ετε
  • αει → ᾳ
    Ejemplo: ἀγαπᾷ ← ἀγαπα-ει
5. La contracción η (ῃ)

Se forma a partir de:

  • εα → η
    Ejemplo: ποιῇ ← ποιε-αι (contraído de ποιε-σαι)

Regla nemotécnica: "El primero gana":

  • Si alfa (α) va primero (αε), se alarga a α.
  • Si épsilon (ε) va primero (εα), se alarga a η (eta, como una "épsilon larga").
6. Contracción miscelánea: οι

Se forma a partir de:

  • οει → οι
    Ejemplo: πληροῖς ← πληρο-εις
    Ejemplo: πληροῖ ← πληρο-ει
7. Contracción de diptongos

Depende de si la vocal contracta y la primera vocal del diptongo son iguales o diferentes:

a. Si son iguales → Simplificación (se elimina una letra repetida)

  • ει ← εει
    Ejemplo: ποιεῖς ← ποιε-εις
  • ου ← οου
    Ejemplo: πληροῦσι ← πληρο-ουσι

b. Si son diferentes → Contracción

  • Si la segunda vocal es iota (ι), se subíndice si es posible.
    Ejemplo: ᾳ ← αει (ἀγαπᾷ ← ἀγαπα-ει)
  • Si la segunda vocal es upsilon (υ) y la letra final del nuevo diptongo es υ, se simplifica a una sola υ.
    Ejemplo: ου ← εου (ποιοῦσιν ← ποιε-ουσι)
8. Los verbos contractos se contraen como si las terminaciones personales fueran las del presente activo de indicativo

Resumen de las Cinco Reglas Principales

  1. ου se forma de εο, οε y οο.
  2. ει se forma de εε.
  3. ω se forma de casi cualquier combinación de ο/ω con otra vocal (excepto regla 1).
  4. α/ᾳ se forma de αε/αει.
  5. η/ῃ se forma de εα/εαι.

Practica con las Reglas de Contracción

Pregunta 1: ¿Qué contracción resulta de ποιε-ομεν?

Pregunta 2: ¿Qué regla explica la contracción en ἀγαπᾷ (de ἀγαπα-ει)?

Pregunta 3: Según la regla nemotécnica "el primero gana", ¿qué contracción resulta de εα?
